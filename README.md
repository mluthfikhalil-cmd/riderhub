# RiderHub 🏍️

All-in-One Mobile App untuk Pemotor Indonesia

## 📱 Tentang

RiderHub adalah aplikasi mobile all-in-one untuk pemotor Indonesia dengan 11 fitur utama:
- Spareparts Marketplace
- Event/Touring Schedule
- Komunitas Rider
- SPBU Finder
- Fuel Tracker
- Battery Health
- Service Cost Calculator
- Second Hand Value
- Motor Diagnostic AI
- Route History
- Personalization

## 🛠️ Tech Stack

- **Frontend:** React Native + Expo
- **Backend:** Supabase
- **Navigation:** React Navigation
- **State:** Zustand
- **UI:** React Native Paper

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run on iOS/Android
npx expo start

# Run on Web
npx expo start --web
```
